Overview

Silicon Motion's portable SSD controller solution is a single-chip solution equips with USB 3.2 Gen2 x 2 interfaces with 20Gb/s bandwidth, with the fully integrated hardware and software solution. It enables the ultra-high speed data transfer between PC and external SSD while enabling new SSD designs to take full advantage of high-density NAND Flash, including the latest generation of 12x-layer QLC NAND.

Because it can interface directly to the host without the need for an external bridge chip, it enables manufacturers of external SSDs to reduce system size and lower their Bill-Of-Materials (BOM) cost.

SMALLER SIZE, MORE STORAGE

The World's Fastest Portable SSD Controller

  • Up to 2,100 MB/s Seq. Read
  • Up to 2,000 MB/s Seq. Write

Features

High Data Integrity and Security
  • Supports Fingerprint Secure
  • Self-encrypting drive (SED) with AES 128/256 and TCG Opal
  • Data retention extension with NANDXtend
Broad host Device Compatibility
  • Compatible with Windows 7 and later version, Mac OS 10.x and Linux kernel v2.4
  • Compatible with USB Type-C Spec. Release 1.3
  • Supports Type-C CC Logi
Best-in-class Low Power Consumption
  • Supports Flash devices operating at VCCQ 1.2V/1.8V
  • Controller core operates from low-power 0.9V supply bus

Technologies

Multi-device Compatibility

The Silicon Motion's portable SSD controllers are compatible with any device that has a USB Type-C connector, including PC, Mac, NB, Tablets, Android devices, smart phones and gaming consoles.

Executive Fingerprint Secure Up

Silicon Motion's portable SSD controllers deliver the highest level of data security by providing a Self-encrypting drive (SED) with AES 128/256-bit encryption, Fingerprint Secure and full compliance with Trusted Computing Group (TCG) Opal specification.

NANDXtend® ECC Technology

Leveraging Silicon Motion's proprietary NANDXtend® error-correcting code (ECC) technology, the Portable SSD controllers enhance the endurance and data retention of 3D NAND, and provide comprehensive protection of data integrity through SRAM ECC and End-to-End data path protection.
