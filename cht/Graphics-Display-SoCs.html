                  簡介

                  慧榮科技 SM768 USB 顯示／擴充座解決方案透過 USB 介面達成多螢幕 4K 超高畫質。目標應用包含 USB 顯示擴充座轉接器，可用於共享辦公室、會議室、教室或旅館。

                  SM768 的 USB 擴充座解決方案，結合高繪圖效能以及低功耗，搭載 CAT™（內容自我調整技術）壓縮技術提供高壓縮率，同時對主機 CPU 降低處理負載。慧榮科技全新開發的獨家技術 InstantView®，擴充座的使用者可以立即將電腦畫面鏡像到任何外接顯示器，完全不需要安裝顯示器驅動程式。

                  慧榮科技採用 SM768 的 USB 擴充座解決方案現已上市並且相容於所有最新版 Microsoft Windows、Apple macOS 和 Linux 作業系統、Android 和 ChromeOS。慧榮科技同時提供參考設計套件做為終端產品設計的簡單設計方案，有助於擴充座製造商縮減開發時間，並縮短新產品設計的上市時程。

                  想看更多 Silicon Motion的影片，請點 SMI的 Youtube頻道。

                  特色

                  • CPU：ARM Cortex R5 處理器 
                  • GPU：128 位元 2D 繪圖引擎支援 HW 遊標、視訊重疊和 Alpha混合
                  • 系統介面：USB 3.0 裝置或 PCI-E Gen2 x1/x2 通道
                  • 單晶片 2x 獨立顯示器頻道
                  • 內容自我調適技術 (CAT) 透過顯示內容壓縮， 加速 USB 顯示
                  • 顯示解析度高達
                    ・4K (3840x2160) @30Hz
                    ・HD(1920x1080) & 2K/全高畫質 @60Hz
                  • 顯示模式：擴充桌面單一及雙重檢視、複製、直向/橫向模式
                  • 內建 1x HDMI v1.4 輸出並支援 HDCP
                  • 雙 CRT 350MHz DAC RGB 輸出
                  • 雙LVDS通道
                  • 雙 24 位元數位影像介面，連接RGB/TTL視訊輸出 
                  • 硬體加速影像解碼器：H.264、MPEG 2/4、MJPEG 等
                  • 提供於單一封裝 (MCM) 嵌入 256MB DDR3 的選項。支援外接 DDR 最高可達 1GB
                  • USB 2.0集線器的4個連接埠可連接周邊設備：鍵盤/滑鼠/網路/儲存裝置
                  • 支援音訊功能的 I2S
                  • 雙 I2C 及 SPI
                  • 通用輸入/輸出：GPIO/UART/JTAG
                  • 兩個 ZV 埠從相機或其他設備擷取傳入的視訊
                  • 軟體支援
                    ・Windows 7/8/10 與伺服器（32 位元和 64 位元）
                    ・macOS X
                    ・Linux 作業系統（Ubuntu、RedHat、SUSE 等） 
                    ・Android
                    ・ChromeOS

                  技術

                  CAT™（內容自我調整技術）

                  慧榮科技的 CAT™（內容自我調整技術）採用多重影像 / 視訊處理演算法壓縮顯示內容來降低所需的頻寬。同時，CAT 以最低的 CPU 使用量來達成低延遲和高畫質顯示。

                  CAT 以自我調整方式處理畫面內容

                  • 圖形／3D影像是透過高壓縮比例 MJPEG 轉碼器處理

                  • 文字是透過慧榮科技所開發的專屬無失真通訊協定進行壓縮

                  • 視訊會以原始壓縮格式傳輸並以晶片加速解碼器解碼

                  CAT 技術將大部分壓縮工作卸載至硬體加速器，不但加快執行速度，而且對 CPU 所施加的負載也比較少。相較於競爭對手產品，慧榮科技CAT技術的負載減少40%至70%。

                  低延遲
                  • SM768 顯示程序管道（壓縮及解壓縮）針對降低延遲進行最佳化
                  • Full HD 畫面的顯示延遲 < 16ms
                  • 下方的實驗室測試影像顯示與 Wireless Miracast 類似的讀數。左邊螢幕為原始顯示，右邊螢幕為 SMI 顯示晶片。兩台螢幕皆複製來顯示相同視訊，畫面計數每秒增加 60 次
                  • 結果顯示兩台螢幕皆隨時顯示相同畫面，代表延遲極低，人眼無法辨識差異

                  USB 轉顯示器橋接的全方位系統解決方案

                  相較於競爭裝置，SMI 繪圖系統單晶片(SoC)提供更低的主機 CPU 負載、完美的使用者體驗。還包含這些相輔相成的特點：CAT 壓縮功能提供的優異圖片品質、雙顯示模式下的低延遲、低耗電量和小巧的尺寸。慧榮科技選用的架構也能讓 OEM 客戶只需幾個外部元件即可設計出完整的擴充座(docking station)和其他產品。

                  USB 和 CAT 軟體以及即時作業系統會在 SM768 之中執行。這說明了此產品能夠支援最新擴充座(docking station)和顯示轉換器 (dongle) 的設計，協助將更大的靈活度、方便性和可攜性帶入家中、辦公室、商店和工廠中的多種運算設備。此外，值得注意的是，相同的 SM768 裝置也支援用於其他類型主機的 PCI-e 介面，讓 OEM 能夠開發同時適用於 USB 市場與其他應用的設計。

                  InstantView®

                  為USB擴充基座提供隨插即用的顯示器鏡映

                  慧榮科技開發出全新的 InstantView 軟體應用程式，可以確實設定在任何顯示器上觀看的主機內容，完全不需要顯示晶片驅動程式。


                  InstantView 軟體是在主機上執行的應用程式。當擴充座透過 USB 連線之後，就會在主機上自動啟動。將已安裝 InstantView 應用程式的電腦連接到擴充座之後，電腦畫面就會出現歡迎訊息。使用者按一下「連線」按鈕，即可將電腦畫面鏡像連接到擴充座上的單或雙顯示輸出。對使用者有影響的是「隨插即用」顯示功能：毫無延遲，也不需要安裝驅動程式或其他軟體。

                  InstantView 軟體支援顯示器即時畫面鏡像。該軟體也支援「延伸顯示」選項。透過 InstantView 軟體以無訊息的方式安裝相關的顯示器驅動程式，使用者即可使用增強的雙顯示功能。無論使用者是選擇鏡像顯示還是選擇「延伸顯示」選項，整個顯示操作都是透過直覺式人性化 InstantView 使用者介面進行控制。如此便不需要 IT 支援人員進行任何介入。 了解更多關於 InstantView
