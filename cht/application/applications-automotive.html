          加速推動車用創新

          隨著科技推動創新，在汽車標準設計要求以及消費者偏好不斷改變需求的情況下，車用市場正在迅速轉型。現代汽車的需求不僅是要求速度，儼然進化成一台會動的資料中心，需要即時處理大量的資料，並且資料儲存的需求也大幅成長，在支援各類汽車應用及娛樂功能的關鍵任務功能方面，NAND 快閃記憶體也將呈現指數成長。

          未來的自動駕駛，對快速、大容量儲存的需求推動嵌入式產品的發展，同時也需要更高階的傳輸系統及更大的儲存單元。

          透過慧榮科技全方位的車用儲存解決方案，將引領加速推動車用產業創新!

          適用於各類車規應用的NAND儲存解決方案

          Silicon Motion憑藉累積超過20年的快閃記憶體儲存設計開發經驗，將其專業知識帶入到車用產業，提供一流的車用儲存解決方案，激發新的可能性。

          專為車用品質要求設計

          車用產業認證

          AEC-Q100 等級 3/2

          ISO 26262 功能安全標準

          供應鏈認證 IATF16949

          ASPICE 認證

          我們所有的車用儲存產品都經過嚴格測試，滿足車用行業及客戶需求，包括符合 AEC-Q100 標準、ISO 26262功能安全標準、IATF 16949供應鏈認證以及 ASPICE 合規軟體開發流程。此外，Silicon Motion 保障車用產品高延展的產品生命週期，可為客戶降低額外的認證成本。

          車用儲存解決方案

          Silicon Motion 提供適用於各類汽車應用的快閃記憶體儲存解決方案產品組合。

          BGA SSD 控制晶片

          • SATA/PCIe NVMe
          • 支援SR-IOV
          • 溫度: -40°C 至 +85°C/-40°C 至 +105°C
          • 容量: 支援 32GB 至 1TB

          eMMC 控制晶片

          • eMMC 4.5/5.0/5.1
          • 100/153 ball-BGA封裝
          • 溫度: -40°C 至 +85°C/-40°C 至 +105°C
          • 容量: 支援 4GB 至 256GB

          UFS 控制晶片

          • UFS 2.1/3.1
          • 車規等級的高耐用性與效能
          • 寫入加速
          • 主機性能增強器
          • 溫度: -40°C 至 +105°C
          • 容量: 支援 32GB 至 512GB

          記憶卡控制晶片

          • 符合 SDA SD3.0/5.0/6.1
          • 車規等級的高耐用性與效能
          • 溫度: -40°C 至 +85°C
          • 容量: 支援 32GB 至 512GB

          FerriSSD

          • SATA/PCIe NVMe
          • 16mm x 20mm BGA
          • 容量: 支援 5GB to 480GB

          Ferri-UFS

          • UFS 3.1/2.2
          • 153-ball BGA
          • 容量: 支援 32GB to 512GB

          Ferri-eMMC

          • eMMC 4.5/5.0/5.1
          • 100/153-ball BGA
          • 容量: 支援 2GB to 512GB

          Graphics SoC/GPU

          • 128 位元 2D 繪圖引擎
          • 顯示解析度高達 4K UHD (3840 x 2160)
          • 溫度: -40°C 至 +85°C
