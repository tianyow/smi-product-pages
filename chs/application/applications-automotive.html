                  加速推动车用创新

                  随着科技推动创新，在汽车标准设计要求以及消费者偏好不断改变需求的情况下，车用市场正在迅速转型。现代汽车的需求不仅是要求速度，俨然进化成一台会动的数据中心，需要实时处理大量的数据，并且数据存储的需求也大幅成长，在支持各类汽车应用及娱乐功能的关键任务功能方面，NAND 闪存也将呈现指数级增长。

                  未来的自动驾驶，对快速、大容量存储的需求推动嵌入式产品的发展，同时也需要更高端的传输系统及更大的存储单元。

                  透过慧荣科技全方位的车用存储解决方案，将引领加速推动车用产业创新!

                  适用于各类车规应用的NAND存储解决方案

                  Silicon Motion凭借累积超过 20年的闪存存储设计开发经验，将其专业知识带入到车用产业，提供一流的车用储存解决方案，激发新的可能性。

                  车用存储解决方案

                  Silicon Motion 提供适用于各类汽车应用的闪存存储解决方案产品组合。

                  BGA SSD

                  • SATA/PCIe NVMe
                  • 支持SR-IOV
                  • 温度: -40°C 至+85°C/-40°C 至+105°C
                  • 容量: 32GB 至 1TB

                  eMMC

                  • eMMC 4.5/5.0/5.1
                  • 100/153 ball-BGA封装
                  • 温度: -40°C 至+85°C/-40°C 至 +105°C
                  • 容量: 4GB 至 256GB

                  UFS

                  • UFS 2.1/3.1
                  • 车规等级的高耐用性与性能
                  • 写入加速
                  • 主机性能增强器
                  • 温度: -40°C 至 +105°C
                  • 容量: 32GB 至512GB

                  Flash Card

                  • 符合SDA SD3.0/5.0/6.1
                  • 车规等级的高耐用性与性能
                  • 温度: -40°C 至+85°C
                  • 容量: 32GB 至512GB

                  FerriSSD

                  • SATA/PCIe NVMe
                  • 16mm x 20mm BGA
                  • 容量: 5GB to 480GB

                  Ferri-UFS

                  • UFS 3.1/2.2
                  • 153-ball BGA
                  • 容量: 32GB to 512GB

                  Ferri-eMMC

                  • eMMC 4.5/5.0/5.1
                  • 100/153-ball BGA
                  • 容量: 2GB to 512GB

                  Graphics SoC/GPU

                  • 128 位 2D 绘图引擎
                  • 显示分辨率高达 4K UHD (3840 x 2160)
                  • 温度: -40°C 至 +85°C
