개요

실리콘 모션은 멀티칩 패키지 솔루션과 낸드 플래시 메모리 컨트롤러의 선도적인 기술력과 경험을 활용한 Single 패키지 스토리지 FerriSSD®와 Ferri-UFS® 와Ferri-eMMC® 제품을 소개 드립니다.

PCIe 4.0 속도 최대화

Ferri SSD Bx 시리즈는 소형 16mm x 20mm BGA 칩 규모 패키지에 PCIe Gen 4 x 4 및 3D NAND 플래시 기술을 자랑하는 임베디드 컴퓨팅 장치용으로 탁월한 스토리지 솔루션입니다.

최대 1TB의 스토리지 용량을 갖춘 이 고성능 스토리지 장치는 최신 SMI 첨단 기술을 최대한 활용하여 6GB/s 이상의 놀라운 순차 읽기 속도와 4GB/s 이상의 순차 쓰기 속도를 제공합니다.

따라서 Ferri SSD Bx 시리즈는 자동차 컴퓨팅, 내비게이션 시스템, 씬 클라이언트, POS(Point-of-Sale) 터미널, 다기능 프린터, 통신 장비, 공장 자동화 도구 및 다양한 서버 애플리케이션을 포함한 광범위한 애플리케이션에 이상적입니다.

업무용이든 개인용이든 Ferri SSD Bx 시리즈는 소형 폼 팩터에 궁극의 스토리지 성능을 제공하여 모든 임베디드 컴퓨팅 장치에 필수적인 구성 요소입니다.

의 FerriSSD® 시리즈는 표준 PCIe NVMe 및 SATA 인터페이스 솔루션을 지원합니다

The FerriSSD® 는 더 빠른 처리 속도, 작고 유연한 폼 팩터 그리고 안정적인 PCIe NVMe / SATA저장 장치를 요구하는 다양한 임베디드 애플리케이션에 최적화하여 설계되었습니다. 업계에서 입증된 컨트롤러 기술, NAND 플래시 및 수동 소자를 하나의 작은 BGA 패키지 안에 조합함으로써 FerriSSD®는 설계에 필요한 수고를 덜고 출시에 걸리는 시간을 감소시키면서도 NAND 테크놀로지 마이그레이션에 대한 우려로부터 보호합니다.

Ferri-UFS® 는 최신 UFS2.1/3.1 표준을 준수하는 다양한 기능의 플래시 컨트롤러와 표준 NAND 플래시 메모리를 결합시킨 고도로 통합된 솔루션입니다. 고성능 스토리지 액세스, 더욱 뛰어난 전력 효율, 간편한 시스템 설계를 통해 Ferri-UFS® 는 자동차, 산업, 임베디드 및 포터블 애플리케이션 분야의 최고의 솔루션이 되었습니다. Ferri-UFS® 의 업계 최저 dppm 관리와 AEC-Q100 인증 및 장기공급지원은 IVI (in-vehicle-infotainment) 와 자용차용 애플리케이션에 최적화되어 있습니다.

Ferri-eMMC® 는 자동차와 다양한 임베디드 애플리케이션에 최적화되도록 설계되었고, JEDEC 표준 eMMC 4.5/5.0/5.1 프로토콜을 준수합니다. 100/153-ball BGA 패키지로 출시되는 Ferri-eMMC® 는 PCB 설계를 용이하게 해주고 생산비용 절감을 가능케합니다.

Ferri 시리즈는 다양한 애플리케이션에 이상적인 NAND 스토리지 솔루션을 제공합니다

기술

Single Root-IO 가상화를 지원하는 Ferri SSD

단일 스토리지에서 여러 VF에 가상 I/O를 직접 지원

SR-IOV 아키텍처는 가상화 개념을 적용합니다. 이 아키텍처는 PCIe 채널을 통해 여러 자동차 모듈을 SSD에 직접 연결할 수 있도록 지원합니다. 이를 통해 지원되는 모든 하드웨어 대상에서 VM을 실행할 수 있는 유연성을 제공하고 사용 가능한 하드웨어 리소스의 사용을 최적화합니다.

가상화 아키텍처의 이점은 더 이상 하이퍼바이저(가상 머신 매니저 또는 VMM)를 통해 데이터 전송을 라우팅할 필요가 없으므로 소프트웨어 코드를 PCIe 형식에서 하이퍼바이저 형식으로 변환했다가 다시 되돌리는 데 따른 지연 시간이 필요하지 않다는 것입니다. 따라서 하이퍼바이저가 실행되는 CPU의 부담도 줄어듭니다.

Benefits:
Cost saving – 여러 대의 저성능 스토리지 장치를 고성능 SSD 하나로 교체할 수 있습니다
Low Latency – SR-IOV 기능은 유연한 가상화 아키텍처에서 네이티브에 가까운 QoS를 제공합니다
Reduced CPU utilization – 가상 머신과 SR-IOV 지원 SSD 사이의 하이퍼바이저 계층을 제거한 덕분에 CPU 효율이 감소됩니다

FerriSSD®: End to End 데이터 패스 보호

SMI의 FerriSSD는 복구 엔진을 통한 완전한 데이터 에러 검출 기능을 포함하고 있으므로, 전체 Host-to-NAND-to-Host 데이터 패스에서 향상된 데이터 무결성을 제공합니다. FerriSSD® 데이터 복구 알고리즘은 하드웨어 (즉 ASIC) 에러, 펌웨어 에러, SRAM, DRAM, 혹은 NAND에서 발생하는 메모리 에러를 포함하여, SSD 데이터 패스에서 발생하는 여러 가지 오류를 효과적으로 검출할 수 있습니다.

FerriSSD®: NANDXtend® ECC 엔진 서비스 수명을 효과적으로 연장

기존의 SSD에서는 NAND shift-read-retry를 사용하는 first-level correction을 수행하기 위하여 표준 BCH 및 RS ECC (error correction coding) 엔진이 사용됩니다. PCI NVMe FerriSSD는 이러한 first-level error correction 외에도 LDPC (Low-density parity check) 코드와 Group page RAID 알고리즘 (고효율 중복 백업)을 사용하는 매우 효율적인 second-level correction 방식을실행함으로써 고객 측의 부품불량률 (dPPM)을 감소시키는동시에 SSD의 사용 수명을 연장시킬 수 있습니다.

데이터 무결성을 향상시키기 위한 IntelligentScan™ 및 DataRefresh™

SMI만의 고유의 IntelligentScan 기능이 활성화되면 호스트의 행위와 작업 환경 (예: 주위 온도)에 따라 셀 블록의 recharge, repair, retire (DataRefresh)를 자동으로 스캔합니다. IntelligentScan과 DataRefresh 를 조합한 결과, FerriSSD®는 일반적인 NAND 규격보다 훨씬 더 오랫동안 사용할 수 있도록 사용 수명이 효과적으로 연장되었습니다.
