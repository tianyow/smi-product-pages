Overview

Unleash the power of the latest 3D NAND with Silicon Motion's family of high-performance SSD controllers and solutions for data center and enterprise infrastructure. Silicon Motion's enterprise SSD controller SoCs are backed by a comprehensive firmware stack, standards-based reference design kits and dedicated support. Maintaining data integrity and security is the highest priority for enterprise. Silicon Motion's controllers provide end to end protection with world-class security, supporting AES-256 encryption and secure boot, while offering full compliance with the Trusted Computing Group's (TCG) Opal specification.

Designed for next-generation data center and enterprise storage, Silicon Motion's enterprise controllers meet the highest standard's of performance, reliability, and security. With customizable turnkey firmware, these SSDs solutions are flexible enough to enable customers to develop application-optimized, differentiated products.

MonTitan™, PCIe Gen5 SSD Development Platform

A high performance, user programmable PCIe Gen5 platform targeting the most challenging data center and Enterprise SSD solutions. Inspired by the OCP Data Center NVMe SSD and NVMe 2.0 specifications, MonTitan™ provides a purpose-built ASIC and FW architecture supporting user SSD Application Optimization in both NVM Express (NVMe) and Host Based FTL usage models for addressing data placement in next generation NAND.

SM8366 - High Performance 16 Channel PCIe Gen5 x4 NVMe SSD Controller

The SM8366 is MonTitan™ platform's high performance, dual ported Enterprise and Data Center PCIe Gen5 x4 NVMe controller targets the most demaning AFA, Caching, HCI, CRM, ERM and traditional DAS and RAID applications. Containing 16 channels and supporting up to 2400MT/s, the SM8366 provides industry leading blazingly fast Sequential (>14GB/s) and Random (>3.0M IOPS) SSD performance. It contains a scalable Single / Dual Channel 40bit DDR4-3200 / DDR5-4800 DRAM interface and is provided in a 21mm x21mm FCBGA package enabling a single chip and FW stack to be used across all new EDSFF form factors.


SM8266 - High Performance 16 Channel PCIe Gen4 x4 NVMe SSD Controller

The SM8266 is a high performance PCIe NVMe Gen4 x4 SSD 16 Channel controller targeting data center applications including durable data storage, JBOF and Cloud applications. Differentiation is provided in Enterprise SSD solutions focused on Performance, Application Acceleration, Reliability and SKU Flexibility. With a complete development platform including a turnkey stack and reference design kit, the SM8266 accelerates development time-to-market for Data Center and Enterprise storage SSDs. The SM8266 maximizes the bandwidth of the PCIe Gen4 x4 Host interface with > 6.5GB/s sequential read and 900K sustained Random Read IOPs. In addition, the SM8266 provides physical and logical isolation with purpose-built hardware and firmware providing consistent, low latency QOS domains.

SM2271 - Providing High Capacity, High Performance SATA III SSD Controller

The SM2271 is a high-performance, 8-channel enterprise SATA controller enabling up to 16TB SSDs with the latest 3D TLC and QLC Flash technology. It fully maximizes the SATA interface supporting 540 MB/s Sequential Reads and 520 MB/s Sequential Writes. Accelerating time to market for lower cost SSDs, the solution includes both ASIC and turnkey firmware.

The SM2271 offers an ideal replacement for lower-performance HDDs while providing more predictable latency than client SSDs. Enterprise SSDs based on a Silicon Motion SM2271 controller combine enterprise-level endurance, security, and performance for read-intensive and mixed workloads, while enabling users to retain existing datacenter infrastructure for the lowest possible total cost of ownership.

Technologies

PerformaShape™

Configured in FW, PerformaShape™ is a mutli-stage shaping algorithm to optimize SSD performance on a per user defined QoS set bases. Combined with using true HW isolation technology, the latest PCIe Gen5 x4 NVMe SSD controller ensures maximum bandwidth performance while maximizing user defined individual performance elements (QOS, Latency, RR/RW, power).

NANDCommand™

The latest PCIe Gen5 x4 NVMe SSD controller maximizes Enterprise NAND functionality by utilizing its 5th generation LDPC engine, machine learning, comprehensive real time media scan, data retention and endurance extension algorithms. These algorithms are efficiently combined with high performance soft-decoding, multi-pass and multi-plane handling, smart buffer management and fine granularity of read/write partitioning for optimized command of next generation low latency TLC/QLC NAND.
